# 1.6 Class of functions

| function |             |
| -------- | ----------- |
| O(1)     | constant    |
| O(logn)  | logarithmic |
| O(n)     | linear      |
| n^2      | Quadratic   |
| n^3      | cubic       |
| O(2^n)   | exponential |
